# QForms

A fullstack framework/platform based on Express and React for building Web UI for databases.

Supports MySql, Postgres, MongoDB.

## CLI Options

```bash
srcDirPath=../name/apps
```

## Make npm module ready for linking

```bash
$ npm link
```

## Build

```bash
$ npx gulp build-dev
```

## Release

```bash
$ npx gulp release
```

## Deploy

```bash
$ npx gulp deploy
```

## Docker

```
$ npm run docker:build
$ npm run docker:run
```
